# Sports News Feed App

A modern Android application that displays sports news with featured matches, latest news, search/filter by category, bookmarks, and a detail view with related stories.

## Features

- **Single Activity Architecture** with Fragment navigation
- **Horizontal RecyclerView** for Featured Matches
- **Vertical RecyclerView** for Latest Sports News
- **Search/Filter** by sport category (Football, Basketball, Cricket)
- **Detail Fragment** with large image, title, description, and related stories
- **Bookmark** functionality using Room database
- **Bookmarks Fragment** to view all saved stories

## Tech Stack

- Language: **Java**
- Minimum SDK: API 24 (Android 7.0)
- UI: RecyclerView, CardView, Material Design
- Database: Room (SQLite)
- Image Loading: Glide
- Navigation: Fragment transactions with back stack
